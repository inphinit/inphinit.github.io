.htaccess file

Apache web server configuration

index.php file

Only change the values of the constants, and only if necessary

server file

Shortcut to start the built-in web server on Linux and macOS

server.bat file

Shortcut to start the built-in web server on Windows

web.config file

IIS web server configuration

public folder

In this folder you can place static files or PHP scripts that will be independent

checkup.php file

{placeholder}, see more details in Check up

system folder

Folder containing your application

dev.php file

It has the same purpose as the main.php script, but it will only work in development mode

errors.php file

It should contain error page settings, such as when a 404 or 405 error occurs, you can call static files or use views, see more details in Error pages
